About Classic Den

This small project was created for The Microsoft Azure Trial Hackathon announnced in February 2022.

My primary goal was to learn a little about the Microsoft Azure services, like Azure Functions or Azure Static Web Apps. It wasn't created to solve any of the world's problems and it wasn't created to offend anyone. Only to bring a little bit of smile and nostalgia for old systems.

What you can see is based on the amazing BOOTSTRA.386 project which is a JS and CSS theme inspired by the 1980s DOS. To learn more about seeing its official GitHub project page or the documentation with a full demo. If you, like me, are amazed at how it looks please support its creator.
